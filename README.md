# 🚀 Oliv Branding Agency UI

A modern and fully responsive branding agency website built using HTML, CSS, and JavaScript.

This project was created as part of Assignment 5 from Cohort 3.0 by Sheryians Coding School. The objective was to recreate a complete agency website as closely as possible while implementing real-world frontend development concepts, responsive layouts, interactive elements, and smooth user experiences.

---

## 📖 Project Overview

Oliv Branding Agency UI is a multi-section landing page designed to showcase a professional branding agency. The project focuses on clean layouts, modern typography, engaging interactions, and responsive design principles.

Unlike previous assignments that contained a single landing page, this project includes multiple sections, reusable components, animations, responsive layouts, and JavaScript functionality to simulate a real-world frontend development workflow.
