# 🚀 Oliv Branding Agency UI

A premium branding agency landing page built using HTML & CSS.

This project was created as part of Assignment 5 from Cohort 3.0 by Sheryians Coding School. The goal was to recreate a modern agency website as closely as possible while applying real-world frontend development concepts, layout structuring techniques, and interactive UI elements.

## ✨ Features Added

### 🔘 Animated Buttons
Custom hover animations created using pseudo-elements and transitions for a modern interactive experience.

### 🖼️ Image Hover Zoom
Project cards, team cards, and blog cards include smooth image scaling effects on hover.

### 🎞️ Infinite Logo Slider
Created an auto-scrolling logo section using CSS keyframe animations.

### 👥 Team Section Hover Effect
Social media icons smoothly appear when hovering over team members.

### 🎨 Premium UI Composition
Carefully structured spacing, typography hierarchy, card layouts, and section balancing inspired by modern agency websites.

### ⚡ Smooth Interactions
Hover states and transitions applied across buttons, cards, images, and navigation elements.
